# christoskataxenos

This is the personal website and blog of Christos Kataxenos. Built with modern web technologies, it serves as a platform to showcase projects, share thoughts, and provide information about my professional background.

## Technologies Used

*   **Next.js:** A React framework for building fast, scalable, and SEO-friendly web applications.
*   **React:** A JavaScript library for building user interfaces.
*   **MDX:** Allows writing JSX in your Markdown content, perfect for interactive blog posts and documentation.
*   **Tailwind CSS:** A utility-first CSS framework for rapidly building custom designs.
*   **next-themes:** For managing light and dark themes.
*   **react-icons & react-social-icons:** For easily including popular icons.
*   **ESLint:** For maintaining code quality and consistency.

## Project Structure

The project is structured to manage various types of content and application features:

*   `src/app`: Contains the main application routes and pages, including specific sections for `bio`, `blog`, and `portfolio`.
*   `src/components`: Reusable UI components used throughout the site.
*   `content/posts`: Stores blog posts and other static content written in MDX.
*   `public`: Static assets like images and favicons.

## Getting Started

To run the project locally, follow these steps:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/christoskataxenos/christoskataxenos.git
    cd christoskataxenos
    ```
2.  **Install dependencies:**
    ```bash
    npm install
    ```
3.  **Run the development server:**
    ```bash
    npm run dev
    ```
    Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

4.  **Build for production:**
    ```bash
    npm run build
    ```
    This command builds the application for production deployment.

# christoskataxenos (Ελληνικά)

Αυτός είναι ο προσωπικός ιστότοπος και το ιστολόγιο του Χρήστου Καταξένου. Δημιουργήθηκε με σύγχρονες τεχνολογίες ιστού και χρησιμεύει ως πλατφόρμα για την προβολή έργων, την ανταλλαγή σκέψεων και την παροχή πληροφοριών σχετικά με το επαγγελματικό μου υπόβαθρο.

## Χρησιμοποιούμενες Τεχνολογίες

*   **Next.js:** Ένα πλαίσιο React για τη δημιουργία γρήγορων, επεκτάσιμων και φιλικών προς το SEO εφαρμογών ιστού.
*   **React:** Μια βιβλιοθήκη JavaScript για τη δημιουργία διεπαφών χρήστη.
*   **MDX:** Επιτρέπει τη συγγραφή JSX στο περιεχόμενο Markdown σας, ιδανικό για διαδραστικές αναρτήσεις ιστολογίου και τεκμηρίωση.
*   **Tailwind CSS:** Ένα CSS framework "utility-first" για τη γρήγορη δημιουργία προσαρμοσμένων σχεδίων.
*   **next-themes:** Για τη διαχείριση ανοιχτών και σκοτεινών θεμάτων.
*   **react-icons & react-social-icons:** Για την εύκολη συμπερίληψη δημοφιλών εικονιδίων.
*   **ESLint:** Για τη διατήρηση της ποιότητας και της συνέπειας του κώδικα.

## Δομή Έργου

Το έργο είναι δομημένο για να διαχειρίζεται διάφορους τύπους περιεχομένου και λειτουργιών εφαρμογής:

*   `src/app`: Περιέχει τις κύριες διαδρομές και σελίδες της εφαρμογής, συμπεριλαμβανομένων ειδικών ενοτήτων για `bio`, `blog`, και `portfolio`.
*   `src/components`: Επαναχρησιμοποιήσιμα UI components που χρησιμοποιούνται σε όλο τον ιστότοπο.
*   `content/posts`: Αποθηκεύει αναρτήσεις ιστολογίου και άλλο στατικό περιεχόμενο γραμμένο σε MDX.
*   `public`: Στατικά στοιχεία όπως εικόνες και favicons.

## Ξεκινώντας

Για να εκτελέσετε το έργο τοπικά, ακολουθήστε αυτά τα βήματα:

1.  **Κλωνοποιήστε το αποθετήριο:**
    ```bash
    git clone https://github.com/christoskataxenos/christoskataxenos.git
    cd christoskataxenos
    ```
2.  **Εγκαταστήστε τις εξαρτήσεις:**
    ```bash
    npm install
    ```
3.  **Εκτελέστε τον development server:**
    ```bash
    npm run dev
    ```

4.  **Δημιουργία για παραγωγή (production):**
    ```bash
    npm run build
    ```
    Αυτή η εντολή χτίζει την εφαρμογή για ανάπτυξη σε περιβάλλον παραγωγής.
