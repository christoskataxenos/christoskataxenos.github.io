# Portfolio Export Settings & Guidelines

Αυτός ο φάκελος περιέχει τις φωτογραφίες για τη σελίδα `/portfolio`. 
Κάθε υπο-φάκελος (π.χ. `american-football`, `football`) διαβάζεται δυναμικά από τον κώδικα και γίνεται αυτόματα ένας νέος "κόμβος" (node) στη σελίδα!

---

## 📸 Ιδανικές Ρυθμίσεις Εξαγωγής (Lightroom / Photoshop / Capture One)

Για να φορτώνει το site αστραπιαία, τα animations να παραμένουν smooth και να μην γεμίσει (φουσκώσει) υπερβολικά το Git Repository:

* **Format (Μορφή):** JPEG (ή WebP αν το υποστηρίζει το πρόγραμμά σου)
* **Color Space (Χρώμα):** sRGB *(Αν βάλεις AdobeRGB τα χρώματα θα βγουν ξεθωριασμένα στους browsers)*
* **Image Sizing (Διαστάσεις):** 
  * Resize to Fit: **Long Edge** (Μεγάλη πλευρά)
  * Pixels: **1920 px** (ή το απόλυτο maximum 2048 px)
  * Don't Enlarge: Check
  * Resolution: 72 ή 300 ppi *(Για οθόνες δεν παίζει κανένα ρόλο, σημασία έχουν μόνο τα 1920 pixels)*
* **Quality (Ποιότητα):** **75% - 80%** 

**Στόχος Βάρους:** Η κάθε φωτογραφία ιδανικά πρέπει να ζυγίζει από **300KB έως 800KB**. Οτιδήποτε πάνω από 1-2MB είναι εντελώς αχρείαστο για το Web (αφού έχουμε βάλει όριο τα 1920px).

---

## 📂 Πώς προσθέτεις νέες κατηγορίες / φωτογραφίες:

1. **Νέα Κατηγορία:** Φτιάξε έναν νέο φάκελο εδώ μέσα (π.χ. `weddings` ή `street-photography`). 
   *Προτίμησε να βάζεις παύλες αντί για κενά στα ονόματα των φακέλων.*
2. **Προσθήκη Φωτογραφιών:** Ρίξε μέσα τις φωτογραφίες σου (`.jpg`, `.jpeg`, `.png`, `.webp`).
3. **Ανανέωση:** Κάνε `git add .`, `git commit` και `git push`. Το site θα διαβάσει αυτόματα τον νέο φάκελο και θα φτιάξει μόνο του το νέο Node στο Portfolio Layout!

*(Σημείωση: Ο κώδικας του Portfolio ψάχνει αποκλειστικά για εικόνες, οπότε αυτό το αρχείο README που διαβάζεις αγνοείται πλήρως και δεν προκαλεί κανένα πρόβλημα).*
