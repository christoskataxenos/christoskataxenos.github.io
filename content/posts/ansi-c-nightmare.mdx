Λοιπόν, επιτέλους ανάσα. Παρέδωσα και τις δεύτερες Γραπτές Εργασίες (ΓΕ2) και νιώθω σαν να βγήκα από χρονοντούλαπο.

Για να είμαστε τυπικοί, στο **Ελληνικό Ανοικτό Πανεπιστήμιο (ΕΑΠ)** έχουν έναν μοναδικό τρόπο να μας κρατάνε σε εγρήγορση (ή σε μόνιμο πανικό, εξαρτάται πώς το βλέπεις). Αυτή η φορά όμως, η εμπειρία είχε κάτι από ταινία επιστημονικής φαντασίας... γυρισμένη στο 1989.

## Ο Ψευδοκώδικας και η Διπολική Διαταραχή

Ας ξεκινήσουμε με το προφανές. Κάποιος πρέπει να μου εξηγήσει τη λογική του να γράφουμε την *ίδια* ακριβώς άσκηση δύο φορές. Μία σε C και μία σε Ψευδοκώδικα.

<Callout type="warning" title="Logic Error">
Το να γράφεις κώδικα που δουλεύει και μετά να πρέπει να τον "χαλάσεις" για να τον μετατρέψεις σε ψευδοκώδικα του ΕΑΠ, είναι μια μορφή βασανιστηρίου που δεν καλύπτεται από τη Συνθήκη της Γενεύης.
</Callout>

Το αστείο είναι ότι πιάνεις τον εαυτό σου να κάνει debug στον ψευδοκώδικα. "Ναι, αλλά εδώ το βελάκι ανάθεσης `<-` είναι σωστό ή μήπως έπρεπε να βάλω άνω-κάτω τελεία;". Λεπτομέρειες που σου τρώνε τη ζωή, ενώ ο compiler της C απλά γελάει μαζί σου.

## ANSI C: Επιστροφή στο Μέλλον

Και πάμε στο "κυρίως πιάτο". Έχουμε 2026. Η τεχνολογία τρέχει, η AI γράφει ποίηση, και εμείς στο ΕΑΠ πρέπει να γράφουμε σε **ANSI C (C89)**.

Ναι, καλά διαβάσατε. Μας ζητάνε να γράφουμε C λες και είμαστε στο Back to the Future με τον Marty McFly. Ξέχνα το να δηλώσεις μεταβλητή μέσα στο `for loop`. Απαγορεύεται. Όλα πάνω-πάνω, νοικοκυρεμένα, όπως το '89.

<Terminal title="gcc terminal">
error: 'for' loop initial declarations are only allowed in C99 or C11 mode
note: use option -std=c99, -std=gnu99, -std=c11 or -std=gnu11 to compile your code
</Terminal>

Ορίστε η διαφορά που μας έκανε τη ζωή δύσκολη:

```c
/* Modern C (Πως θα έπρεπε να είναι) */
for (int i = 0; i < 10; i++) {
    printf("%d", i);
}

/* ANSI C (Πως το θέλει το ΕΑΠ) */
int i; /* Πρέπει να δηλωθεί στην αρχή του block αλλιώς τρως error */

for (i = 0; i < 10; i++) {
    printf("%d", i);
}
```

## Η Σωτηρία της Python

Ευτυχώς, η δεύτερη εργασία (της ενότητας ΠΛΗΠΡΟ) είχε Python. Μετά από 48 ώρες `malloc`, `structs` και "segfaults" στην ANSI C, το να γράφεις Python έμοιαζε με το να βγάζεις τα στενά παπούτσια μετά από 10 χιλιόμετρα περπάτημα.

Εκεί που στην C ήθελες 50 γραμμές κώδικα για να διαβάσεις ένα αρχείο, στην Python το έκανες σε 3. Η διαφορά στην ψυχική υγεία είναι μετρήσιμη σε κούπες καφέ.
